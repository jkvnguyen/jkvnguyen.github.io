Hermite Spline


What I did:

Implemented a class that generated hermite splines with maximum 40 control points. Initialized Catmull-Rom properties to interactive spline creation where clicking on the user interface adds a point. Added manipulation of control points and tangents, arc length parameterization, and input/output from/to a file.

Source code (Running the source code)

Class Methods (Using the Shell)

Manipulation of control points and tangents:
  • system {class name} set tangent {index} {x y z}
  • system {class name} set point {index} {x y z}
  • system {class name} add point {x y z sx sy sz}
    # adds a point with the given tangent at the end of the spline

Arc length parameterization using a piecewise linear approximation and look up table.
  • system {class name} getArcLength {t}

Input/output from/to a file:
  • system {class name} load {file name}
  • system {class name} export {file name}

    • # Format for the spline file:
    {Spline Name} {n}
    {p1[x] p1[y] p1[z] s1[x] s1[y] s1[z]}
    {p2[x] p2[y] p2[z] s2[x] s2[y] s2[z]}
    ….
    {pn[x] pn[y] pn[z] pn[x] sn[y] sn[z]}

    where the prefix “p” refers to control points and the prefix “s” to tangents.
