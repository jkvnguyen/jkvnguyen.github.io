summary

A personal project to familiarize myself in Unreal Engine 5, Windrunner is a stealth survival game where the objective is to gather materials and escape a prison camp.

Game is inspired and based on Brandon Sanderson’s Stormlight Archive’s The Way of Kings.

takeaways

While school and smaller scale professional projects used version control, it had always made me nervous when using it. Because of the sheer number of people on the project there is always potential for merging conflicts. Working on this project, I became more comfortable with using Git and communication with team members. In a team setting, communication is key when resolving conflicts but it is also key to understand another’s work process to avoid conflict and avoid lost time.
