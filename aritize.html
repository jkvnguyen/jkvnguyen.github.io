ARitize


What I did:


Developed the UI/UX for an Augmented Reality mobile platform designed to increase brand awareness, customer engagement, and online sales.

  • Researched a variety of different social media and Augmented Reality apps and applied successful design trends to be used as part of designing ARitize
  • Provided detailed explanations for onboarding Augmented Reality experiences for users who were unfamiliar with AR along with the app
  • Designed pages for: home, subscriber profile, user profile, the Augmented Reality experiences, categories/featured, search filter, QR Scan & Code protection
  • Developed a clickable prototype for team members to use and give feedback on

Brainstorming + Design Goals


Before these designs, ARitize was simply an app that only held Augmented Reality experiences without an organized system to display the experiences and mostly served as a proof of concept. Experiences were categorized only by brands but this was not the overall vision the company had envisioned and aims for the app would begin as:

  1. The Instagram of AR - a social media platform for everyone to showcase AR
  2. Brand + Portfolio Showcase - Target audience would be for brands and developers hoping to showcase their products and portfolio to share
  3. Live streaming service - Users would be able to stream AR content live as a hologram and experience live streams as virtual spectators
  4. Promotional and QR codes that could be entered and scanned to join specific channels and experiences
  5. Location based QR/promotional code to be scanned in person and on live products to be used for discounts
  6. Channel/profiles hold Augmented Reality experiences
  7. Each experience is categorized by a tag and category to be used by a search engine

With these ideas and intentions for the app established, the design goals would include:

  • Create an onboarding experience that describes the app and educates them about AR
  • Provide extensive information on AR use such as lighting and plane detection to properly enable AR
  • Develop a channel system that is easy to use and navigate - subscription/following based
  • Add menu items that are consistent with other social media platforms
  • Create a QR scan and Code page that is consistent with design trends

Menu Bar


The menu bar consists of top and bottom component to create symmetry and act as containers to hold clickable buttons. The top component contains icons for a search, app logo, and profile. The bottom component contains a home, code, and scan icons.

Buttons:

  • A Search button that goes to a search page revealing featured content and the ability to search specific content
  • Home button that goes to the default page when the app is loaded. Home page contains content for user's current followings/subscription showing if new content is available or if a channel is currently doing a live stream
  • Code button that prompts a "enter a code" text prompt for users to enter a code. Entering the correct code will lead users to private content that cannot be indexed via searched or by normal subscription style format. Pages for entering a code are either the AR experience itself or the channel of the AR experience
  • Scan button is the same as the code button.
  • Profile button goes to a profile page displaying the list of the user's account information such as email and ability to change password and logout. Profile page also includes options for viewing user's channel own AR content and the list of channels that user is subscribed too.
  • ARitize logo is present simply as a branding style to promote the app more publicly

Home Page


The default page when the app is first loaded. It contains a list of all content from the list of followings/channels the user is currently subscribed to. The page is formatted in a two column format because it best displays the channel's logo increasing brand awareness. Each channel is designed in its own block with a square logo and name underneath. Channel and channel text are all clickable and goes to the channel page revealing all content of that channel. Live/new content will be highlighted with a small banner on the home page showing 'New' or 'Live'. Overflowing channels will simply be scrollable.


Channel Page


Channel page holds content for a channel's page and designed with a small profile pic and banner image (used to promote promotions) at the top as a menu top bar. Additional text is displayed below the profile pic giving the user a way to provide context to their channel and promote any promotional text or events. Content is assorted in a one column display and AR experiences are presented in individual blocks with preview for the AR experience.


AR Experience


The AR experiences all remains the same with the exception of sequential onboarding text giving detailed explanations of how to setup the experience and use AR and some additional UI buttons for additional info, help, and share.

Text for the AR:

  1. Ensure there is sufficient lighting and that you are pointing your device on a flat surface.
  2. (Once flat service detected) Tap to place.

Additional info simply shows any additional text that exists for the experience by the channel. Help provides additional information on how to use and setup the experience. Sharing sets a screenshot of the experience and sends it via a variety of social media platforms.

Search Page


Search page filters the result of the text inputted and lists experience with the same or similar text input. Search displays the searched elements in the same manner as the profile page showed in a single column and experiences are listed out within that column.
