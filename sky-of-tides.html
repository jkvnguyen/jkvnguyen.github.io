summary

Sky of Tides is a sci-fi narrative game set in a technologically advanced universe on the brink of war. Play as Rin, who must search for her missing father and heal the broken planet of Numen. Explore the world while your choices shape your character.

Worked freelance as a UI Developer, where I developed many of the game’s UI systems and assisted with UX design.








what i did:




01. UI Development


Update and added game UI elements for game menus, character screens, and events throughout the game.




02. UX Design


Collaborated with creative team to feature improved user feedback and consistent design choices.




03. UI Animations


Incorporated sprite animations to events in game and menu screens.




04. Notifications System


Implemented a system to notify players of newly found unlockables.




05. Technical Writing


Documented and organized the team GDD and CMS.




06. QA & Debug


Playtested and debugged in-game inconsistencies, bugs, and functions.

takeaways

While school and smaller scale professional projects used version control, it had always made me nervous when using it. Because of the sheer number of people on the project there is always potential for merging conflicts. Working on this project, I became more comfortable with using Git and communication with team members. In a team setting, communication is key when resolving conflicts but it is also key to understand another’s work process to avoid conflict and avoid lost time.
