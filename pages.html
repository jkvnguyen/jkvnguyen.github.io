About

A web application for an online book store. Main components of the application includes: data access component, product catalog service, order process service, session controller, book store main page, shopping cart page, payment page, analytics page, web services, performance and scalability, security, and a design document.

Responsibilities: Application design, front-end, back-end, technical writing
Project length: 6 weeks

Source code


What I did:

  • Designed store's promotional pages, books, and main pages.
  • Contributed to the application's database and architecture designs.
  • Implemented search bar functionalities and search filtering.
  • Documented all design strategies, UML case diagrams, class diagrams, and sequence diagrams.
  • Implemented AJAX functionalities.


Main Page

The main page is built with a simple design and display of content. Login options are located on the top right where new users are registered onto the database while existing users are retrieved. Menu bar features home button and browsing of categories. Search bar allows for browsing of author, book name, and book genre. Shopping cart is located on the top right and leads to a shopping cart page where users can make a payment to purchase an item. Promotional pages give the website depth and are add-ons to project specifications. Top sellers are featured as an add-on and can be clicked to view these books and its information as well as analytics (simply being a top seller).


Search Page

The search page is queried by the user based on author, book name, and book genre. Available books based on the query will be shown. Filter options can found on the left side of the books displayed and help browse books based on certain specifications. Filters include only genre and prices.




Books can be clicked upon to see additional information about individual books.
