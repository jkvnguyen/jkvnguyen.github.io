Diablo 2 Resurrected: UI Mockup (Personal Project)



Role: UX Designer

Skills: OpenGL, Javascript, C++

Platform: PC


Summary

A personal project to improve Diablo 2 Resurrected's lobby system.


What I Did:

  • Raycaster: Extended a ray casting program; completing transformation and lighting functions of 3D shapes. Implemented a cylinder intersection function and added textures to 3D shapes.
  • Spaceship: Extended a program to steer a 3D spaceship by applying a series of affine transformations. Added interactivity where holding down left click while moving the mouse moves the camera such that the user can observe the ship from multiple directions.
  • Hermite Spline: Implemented a class that generated hermite splines with maximum 40 control points. Initialized Catmull-Rom properties to interactive spline creation where clicking on the user interface adds a point. Added manipulation of control points and tangents, arc length parameterization, and input/output from/to a file.
